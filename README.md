## Installation

### - Laravel

### Install Dependencies

```
composer install
```

### Run application local

```
php artisan serve
```

### - NextJs

### Navigate to front end folder

```
cd resources/js
```

### Install Dependencies

```
npm install
```

### Run application local

```
npx next dev
```

### Convert SCSS to CSS

```
sass resources/js/public/assets/css/style.scss:resources/js/public/assets/css/style.css
```
